# 🔑 鍵受注管理システム

鍵の受注状態をチームで共有・管理するWebアプリです。

## 機能

- **5段階ステータス管理**: 受注 → 手配済み → 入荷済み → 作業アポ済み → 完了
- **件数バッジ表示**: 各ステータスの件数をリアルタイム表示
- **ステータス遷移**: 各受注から適切なステータスへワンクリック移行
- **顧客情報管理**: 氏名・マンション・部屋番号・電話・作業内容・金額
- **検索**: 氏名・マンション・電話番号で絞り込み
- **Google Sheets連携**: バックアップ・チーム共有
- **ローカルストレージ**: オフラインでも動作

## セットアップ

```bash
npm install
npm run dev
```

## ビルド & デプロイ（GitHub Pages）

```bash
npm run build
# dist/ フォルダを GitHub Pages で公開
```

## Google Sheets 連携

`GAS_SETUP.md` を参照してください。

## 技術スタック

- Vanilla CSS（カスタムプロパティ）
